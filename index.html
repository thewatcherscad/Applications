FIRST-TIME SETUP
Connect your JSONBin database
Step 1 — Create your JSONBin
You need a free JSONBin account to store all data permanently across devices.
1. Go to jsonbin.io and create a free account
2. Click + CREATE BIN and paste this as the content:

{"forms":[],"applications":[],"users":[],"activity":[]}

3. Click CREATE BIN — copy the BIN ID from the URL
4. Go to API Keys in your account → create a key → copy it
These are saved in your browser and never sent anywhere except JSONBin.
